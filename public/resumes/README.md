# Resume Files

Place your PDF resumes in this folder:

1. **UI/UX Resume** → Save as: `ui-ux-resume.pdf`
2. **Full Stack Resume** → Save as: `full-stack-resume.pdf`

Example path:
```
public/resumes/ui-ux-resume.pdf
public/resumes/full-stack-resume.pdf
```

The buttons will automatically link to these files once added.
